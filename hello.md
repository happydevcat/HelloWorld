## gitt 명령어 요약


- clone : 원경 저장소 복사
- add : 스테이지 영역에 작업파일 추가. 
- commit : save 스테이 영역의 파일대상으로 commit 을 만들수 있다.
- push : 원격 저장소에 커밋을 업로드한다.
